# Classes and Object Model

Object-Oriented Programming in Python: define classes, create instances, manage state, implement methods, and use special methods (`__init__`, `__str__`, `__repr__`).

## Tasks

- **0. My First Square** — `0-square.py`: Define an empty class `Square` (no attributes or methods).
- **1. Square with Size** — `1-square.py`: Class `Square` with private instance attribute `__size` set in `__init__`.
- **2. Size Validation** — `2-square.py`: Validate `size`: must be int (TypeError) and >= 0 (ValueError); default size=0.
