# Abstract Classes & Interfaces

ABC, abstract methods, interface-like design, duck typing, mixins.

## Tasks

- **0. Abstract Animal** — `animals.py`: Abstract class `Animal(ABC)` with abstract `sound()`; `Dog` returns "Bark", `Cat` returns "Meow".
- **1. Shapes, Interfaces, Duck Typing** — `shapes.py`: Abstract `Shape` (area, perimeter); `Circle`, `Rectangle`; `shape_info(shape)` by duck typing.
