# Core Data Structures

Lists, tuples, sets, dictionaries: iterate, access, and manipulate without mutating when required.

## Tasks

- **0. Print a list of integers** — `print_list_integer.py`: print all integers of a list, one per line.
- **1. Safe access to a list element** — `element_at.py`: return element at idx, or None if idx negative or out of range.
- **2. Replace an element in a list** — `replace_in_list.py`: replace element at idx; if idx invalid, return list unchanged.
- **3. Print a matrix of integers** — `print_matrix_integer.py`: print 2D list, one row per line, values separated by space.
- **4. Tuple addition** — `add_tuple.py`: return new tuple of two ints (element-wise sum); missing=0, ignore beyond first two.
