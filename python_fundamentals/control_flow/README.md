# Control Flow

Python control flow: conditionals (`if`, `elif`, `else`) and loops.

## Tasks

- **0. Positive anything is better than negative nothing** — `positive_or_negative.py`: print whether a random number is positive, zero, or negative.
- **1. The last digit** — `last_digit.py`: print the last digit of a random number and whether it's greater than 5, zero, or less than 6 and not 0.
