# Functions & Modularity

Functions, return vs print, modules, and `if __name__ == "__main__"`.

## Tasks

- **0. islower** — `islower.py`: function that returns True if character is lowercase (ASCII), False otherwise.
- **1. To Uppercase** — `uppercase.py`: print string in uppercase (ord/chr), no .upper(), no return.
- **2. Print Last Digit** — `print_last_digit.py`: print and return last digit (always positive).
- **3. a ^ b** — `pow.py`: return a**b without ** or imports, using a loop.
- **5. Import a Simple Function** — `add_0.py`: defines add(a,b), imports it in __main__, prints 1 + 2 = 3; no execution when imported.
