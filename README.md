# holbertonschool-core-engineering

Repositorio para ejercicios de Python del programa Core Engineering (Holberton).

## Estructura de trabajo

- **En este repositorio**: solo lo que pide cada ejercicio (código, archivos requeridos, tests si los piden).
- **Fuera del repositorio**: análisis, notas, documentación extendida y pruebas adicionales (por ejemplo en `hbtn-betatest/` o la carpeta que uses como workspace).

## Organización de ejercicios

Cada ejercicio puede ir en su propia carpeta, por ejemplo:

```
holbertonschool-core-engineering/
├── 0x00-python-hello_world/
├── 0x01-python-if_else_loops_functions/
├── ...
└── README.md
```

O según la estructura que indique el proyecto/checker de Holberton.

## Cómo usar

1. Crea la carpeta del ejercicio dentro del repo.
2. Añade únicamente los archivos que pida el enunciado.
3. Haz commit y push de los cambios.
4. Guarda análisis, apuntes y pruebas en una carpeta externa (por ejemplo `../analisis-core-engineering/` o dentro de `hbtn-betatest/`).
